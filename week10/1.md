# week10-1-signal-7

## Условие

Программа считывает со стандартного потока ввода 64-битные целые беззнаковые числа и вычисляет их сумму. Однако, если после чтения очередного числа в течение 100 мс (миллисекунд) следующее число не было считано, к сумме прибавляется 1000. Если после этого ещё через 100 мс число не было считано, прибавляется 2000, затем через 100 мс прибавляется 3000 и так далее. Успешное чтение числа сбрасывает таймер и прибавляемое значение в начальное состояние. Момент начала работы программы рассматривается как чтение числа, то есть время начинает отсчитываться от начала работы программы.

На стандартный поток вывода напечатайте итоговую сумму.

Для обработки тайм-аутов используйте интервальный таймер. Обработчик сигнала должен только устанавливать флаг, никакие другие действия в обработчике таймера не допускаются.

*Language: gcc-GNU C 15.2.0
