# week11-2-socket-2

## Условие

На стандартный поток ввода подаются текстовые строки, не превышающие в длину 1000 символов. Каждая текстовая строка содержит имя хоста HOST и имя сервиса SERVICE, каждое из которых является символьными строками.

Для каждой пары HOST SERVICE на стандартный поток вывода напечатайте адрес IPv4 и номер порта.

Если имя HOST отображается в несколько IP-адресов, выводите IP-адрес, представление которого в локальном порядке байт (32 бита) минимально.

Если при отображении возникла ошибка, на стандартный поток вывода напечатайте текстовое сообщение об ошибке.

Используйте функции getaddrinfo, gai_strerror, freeaddrinfo, inet_ntop, ntohs, ntohl.

**Примечание.** Текущий IP-адрес хоста google.com может отличаться от приведенного в примере.

## Примеры

### Пример 1

<span style="color: gray; font-weight: bold;">Входные данные</span>
```
google.com https
go.go 25
```

<span style="color: gray; font-weight: bold;">Результат работы</span>
```
173.194.32.128:443
No address associated with hostname
```
